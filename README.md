# Campus Security

A full-stack incident management platform designed for campus safety dispatch teams to track incidents, manage dispatch workflows, and generate structured shift handoff reports.

Built with **Next.js** on the frontend and a **Flask + MongoDB** backend. Deployed on **Vercel** and **Render**.

## Features

- Incident creation, dispatch tracking, and resolution workflows
- Server-side filtering by status, priority, and timeframe
- Editable incident side-panel interface
- AI assisted dispatcher shift handoff summaries
- AI assisted incident autofill based on description
- AI assisted high/critical emergency response generation (SMS, email)
- Dashboard analytics with cached stats and incident statistics
- Secure authentication via HTTP-only session cookies
